# Support

## Getting Help

If you need help with MeridianAlgo projects, use the channels below to get the fastest response.

## Recommended Channels

- GitHub Issues: Use this for bug reports and feature requests
- GitHub Discussions: Use this for questions, ideas, and community help
- Email: Use this for private inquiries

## Where to Ask What

### Bugs

- Create an issue in the relevant repository
- Include steps to reproduce, expected behavior, and actual behavior
- Attach logs or screenshots when relevant

### Feature Requests

- Create an issue in the relevant repository
- Explain the problem you are trying to solve and what success looks like

### Security Issues

- Do not open public issues for security vulnerabilities
- Follow our Security Policy: see SECURITY.md
