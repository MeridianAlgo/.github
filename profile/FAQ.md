# FAQ

## What is MeridianAlgo?

MeridianAlgo is a nonprofit organization building open-source technology and education initiatives.

## Where are the actual projects?

This repository is a profile-style hub. The main projects live across MeridianAlgo public repositories listed in README.md.

## How do I contribute?

Start with CONTRIBUTING.md and then open a pull request in the repository you want to improve.

## How do I report a security issue?

Do not open a public issue. Follow SECURITY.md and email security@meridianalgo.org.

## How do I request a new project or initiative?

Open an issue or discussion in the relevant repository describing the problem, the audience, and success criteria.
