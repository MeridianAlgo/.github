# MeridianAlgo
**Empowering the world with Open-Source Innovation**

A nonprofit organization active in 4 countries, creating technology for students, companies, and everyone in between. All our work is 100% open-source, transparent, and community-driven.

## What We Do
- **Financial Literacy Workshops**: Interactive sessions on budgeting, investing, and debt avoidance
- **Open-Source Tools**: Open-source technology development
- **Community Partnerships**: Collaboration with schools, youth groups, and NGOs

## Projects

We have a growing collection of open-source projects focused on financial technology, AI, quantitative finance, education, and blockchain. Here are **all our current public repositories**:

### Core Tools & Platforms
- **[AraAI](https://github.com/MeridianAlgo/AraAI)** — AI-powered financial analysis platform for stock volatility forecasting, market predictions, and portfolio optimization.
- **[FinAI](https://github.com/MeridianAlgo/FinAI)** — In-house LLM research focused on finance-based chats and requests.
- **[Cryptvault](https://github.com/MeridianAlgo/Cryptvault)** — Professional-grade cryptocurrency analysis with advanced AI/ML predictions, 50+ pattern recognition, and terminal charts.
- **[FinDB](https://github.com/MeridianAlgo/FinDB)** — Multi-source financial data scraper and database with daily automated updates.
- **[Midnight.AI](https://github.com/MeridianAlgo/Midnight.AI)** — Multi-objective trading engine with pretrained model, direct Alpaca API paper trading, and built-in backtester.

### Education & Learning
- **[Learn-Quant](https://github.com/MeridianAlgo/Learn-Quant)** — Beginner-friendly utilities and examples that power our quantitative finance programs in Python & JavaScript.

### Packages & Libraries
- **[Python-Packages](https://github.com/MeridianAlgo/Python-Packages)** — Official Python packages (install via `pip install meridianalgo`).
- **[Javascript-Packages](https://github.com/MeridianAlgo/Javascript-Packages)** — Official NPM packages (install via `npm install meridianalgo`).

### Utilities & Templates
- **[Pine-A-Script](https://github.com/MeridianAlgo/Pine-A-Script)** — Open-source transpiler that converts TradingView Pine Script (v5 & early v6) to clean, executable JavaScript.
- **[No-Ticker-Left-Behind](https://github.com/MeridianAlgo/No-Ticker-Left-Behind)** — Global stock ticker database that generates a regularly refreshed stock universe.
- **[Apex-Analysis](https://github.com/MeridianAlgo/Apex-Analysis)** — Beginner-friendly stock analysis and research platform (public template).
- **[Basic-Sentiment-Analysis](https://github.com/MeridianAlgo/Basic-Sentiment-Analysis)** — FinBERT-powered sentiment analysis for financial news (public template).

### Blockchain & Crypto Infrastructure
- **[AstryxChain](https://github.com/MeridianAlgo/AstryxChain)** — Quantum-walk-inspired cryptographic hash function built for blockchains and post-quantum security.
- **[LuminaChain](https://github.com/MeridianAlgo/LuminaChain)** — Lightweight, sovereign Rust L1 blockchain purpose-built for unbreakable stablecoins (public template).

### Other
- **[meridianalgo.org](https://github.com/MeridianAlgo/meridianalgo.org)** — Source code for our official website.
